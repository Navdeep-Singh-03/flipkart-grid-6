# Flipkart Grid 6.0 — Traffic Demand Prediction

Built an ML pipeline for traffic demand forecasting using Bengaluru CCTV/sensor data.

## Approach
- Ensemble of LightGBM, XGBoost, and CatBoost
- Geohash-based target encodings for spatial features
- Achieved OOF R² of ~0.9974
